object_tools API¶

The object_tools provides the methods to create and validate Ook types and objects.

Usage¶

There are two basic operations provided by object_tools; schema definition and object handling. Schema definition entails type creation and validation.

Schema Tools¶

To validate a schema definition, utilize the validate_schema() method.

Schema are composed of PropertySchema objects. If there is a need, individual PropertySchema objects can be validated individually with the validate_property() method.

To create a python type for a given SchemaType utilize the create_ook_type() method. Ook object instances created by a generated type are child classes of the object_types class.

Object Tools¶

Ook objects created by either subclassing BaseType or via create_ook_type(), will need to be validated. Utilize the Ook object validate_object() method for validation.

If the need should arise for validation of an Ook object by value, utilize the validate_value() method.
